OOP Design (Hangman)

  1. Understand the Problem
    • Set up game board
    • Game loop
    • Draw board
    • Generate a word
    • Get letter from player
    • Validate letter
    • Check if letter id correct or wrong
    • Finish game if whole word was guessed

  2. Pseudocode the High Level Approach
    3.   Set up game initially
      Create game
      Create players
  Start the game loop
      Render the board
      Generate a word
      Display word length
      Get letters from players
      If letter is correct
        Display letter on line
      Else
        Draw stick figure body parts
        Display wrong letters
      If game should end
        Display victory/loss message
        Stop looping
      Else
        Keep looping

Start the game loop [Hangman]

    Else

  4. Make Javascript-ish Pseudocode
    5. class Hangman
    initialize
        set up the board
        set up the players
        assign starting player

    play
        loop infinitely
            call the board rendering method
            generate a word
            display word length
            get letters from players
            validate letters

    check_game_over
        check_victory
        check_loss

    check_victory
        if board says the word is correct
            display victory message

    check_loss

            display loss message
