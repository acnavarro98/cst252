Lab 4: Pseudocoding and Problem-solving

For this lab, the main idea is to get in the habit of pseudocoding before starting to code. We created pseudocode for an everyday task and for a computer task.

Challenges

We were able to come up with an everyday task very quickly. Although, we did have trouble coming up with a computer task.

Results

Here are the pseudocodes we created:

Making Lemonade:

  1. Cut lemons in half
  2. Squeeze lemons in a pitcher
  3. Add water
  4. Add sugar
  5. Mix
  6. Add ice or refridgerate
  7. Pour into cup
  8. Sip and enjoy!

Converting meters to feet:

  1. Input meter value (M)
  2. Calculate feet(F):
    • Multiply M by 3.2808
  3. Output the value of F
  4. Stop
